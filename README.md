# coLab 实验室管理系统
项目简介：

实验室管理系统可以方便管理记录实验室相关信息，实验室负责人创建实验室后可以对实验室的信息进行维护，功能模块主要有：1实验室财产管理；2实验室成果展示（论文，比赛等）；3实验室活动

对于新加入实验室的人可以很快了解到实验室相关的历史信息等，方便大家组织活动，记录实验生活，了解前人的工作。

在线体验地址： http://120.78.78.15:8082/  账号 admin 密码 12345


## 环境搭建要求

* JAVA_VERSION: 1.8
* Mysql: 5.7 及以上
* Redis: 5.0.14
* IDE: IDEA

### 2. 如需要，设置JDK等必要设置
