# coLab 实验室管理系统


## 环境搭建要求

* JAVA_VERSION: 1.8
* Mysql: 5.7 及以上
* Redis: 5.0.14


## 项目运行步骤

| 注： 本项目已经集成flyway 数据库版本控制框架，无需创建数据库等操作。

### 1. 下载源代码或者在IDEA中通过Github代码仓库创建项目

### 2. 如需要，设置JDK等必要设置

### 3. 等待 maven 加载

### 4. 直接运行项目

### 5. 输入 localhost:8082 访问首页

### 6. 点击登录， 输入账号 admin  密码 12345 登录即可体验项目
